# ROI Calculator

FacturaGuard v3.3 adds a public ROI calculator.

## Page

```txt
/roi
```

## Purpose

The calculator helps during sales demos and customer discovery.

It estimates:

- monthly invoices
- manual hours
- manual cost
- estimated saved hours
- estimated saved cost
- net monthly value
- annual net value
- monthly ROI

## Default assumption

The calculator assumes FacturaGuard reduces 55% of the manual operational time spent on e-Factura checks, status tracking, reporting and document lookup.

This is a sales/discovery estimate, not a guaranteed result.

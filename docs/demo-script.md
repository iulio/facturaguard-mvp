# FacturaGuard Demo Script

## Duration

10-15 minutes.

## Suggested flow

1. Open `/landing`.
2. Explain the problem: e-Factura status tracking is fragmented and reactive.
3. Open `/pricing` and `/roi`.
4. Login and show the portfolio dashboard.
5. Open `/work-queue` and filter urgent/rejected invoices.
6. Show invoice metadata and notes.
7. Show documents and audit log.
8. Show client portal.
9. Show API keys and `/developer`.
10. Show `/system-status` to prove operational maturity.

## Core pitch

FacturaGuard helps Romanian accounting offices and SMEs monitor e-Factura operational risk before it becomes a compliance or client problem.

## Current demo caveats

- ANAF connector is mock.
- NETOPIA payment flow is mock.
- Production deployment needs real secrets, backups, object storage and legal validation.
